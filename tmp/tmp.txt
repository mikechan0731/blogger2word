<h3 class="post-title entry-title" itemprop="name">
以設計思考為核心的創新思維課程
</h3>
<div class="post-header">
<div class="post-header-line-1"></div>
</div>
<div class="post-body entry-content" id="post-body-3891111303920736382" itemprop="description articleBody">
<span style="font-size: large;">「創意沒有極限</span><br/>
<span style="font-size: large;">如果有極限</span><br/>
<span style="font-size: large;">就是思維被框住了」</span><br/>
<span style="font-size: large;"><span style="font-size: small;"><br/></span>
<span style="font-size: small;"><br/></span>
<span style="font-size: small;">上完</span></span><br/>
<span style="font-size: large;">以設計思考為核心的</span><br/>
<span style="font-size: large;">創新思維課程後</span><br/>
<span style="font-size: large;"><span style="font-size: medium;"><br/></span>
</span><br/>
<a name="more"></a><span style="font-size: large;"><span style="font-size: medium;"><br/></span>
<span style="font-size: medium;">摘錄幾位學員課後的收穫回饋:</span></span><br/>
<span style="font-size: large;"><span style="font-size: medium;"><br/></span>
</span><br/>
<ul>
<li><span style="color: blue; font-size: large;">很實用! 可套用在發想各種新點子, 我會用來檢視發展自己的創意</span></li>
<li><span style="color: blue; font-size: large;">思考方向, 思維提升, 結合生活上周邊事物</span></li>
<li><span style="color: blue; font-size: large;">如何激發創意? 有系統的方法!</span></li>
</ul>
<ul>
<li><span style="color: blue; font-size: large;">同理心做到洞察, 優化做到換位思考, 全新觀點到全方位</span></li>
<li><span style="color: blue; font-size: large;">創新必須開放的心態, 多元的成員組合可以激發想法, 透過方法論進而發展出一個新的原型</span></li>
<li><span style="color: blue; font-size: large;">改變思考方式, 不要直接給答案, 集合大家的想法, 發散, 收斂, 選擇, 優化 </span></li>
</ul>
<ul>
<li><span style="color: blue; font-size: large;">設計沒有那麼難, 從產品思維調整為客戶導向思維</span></li>
<li><span style="color: blue; font-size: large;">從日常生活的人事物, 去洞察需求, 改善流程, 改善生活, 發想, 優化, 原型, 測試修正</span></li>
<li><span style="color: blue; font-size: large;">有結構脈絡的學習, 創意的發想, 創新作法的歸納, 態度+努力+團隊合作, 在公司實際案例演練中, 去實踐驗證設計思考</span></li>
</ul>
<div>
<span style="font-size: large;"><br/></span></div>
<span style="font-size: large;"><span style="font-size: medium;"><br/></span>
<span style="font-size: medium;"><span style="font-size: small;"><br/></span>
<span style="font-size: small;"><br/></span>
<span style="font-size: small;">從客戶的痛點開始</span></span></span><br/>
<span style="font-size: large;">---&gt;</span><br/>
<span style="font-size: large;">訪談(運用訪談的三個原則)</span><br/>
<span style="font-size: large;">深度訪談 (關於這個狀況, 你可以再多描述一點嗎?)</span><br/>
<span style="font-size: large;">洞察需求</span><br/>
<span style="font-size: large;">重新定義問題</span><br/>
<span style="font-size: large;">創意發想發散</span><br/>
<span style="font-size: large;">創意收斂</span><br/>
<span style="font-size: large;">點子優化</span><br/>
<span style="font-size: large;">實作Prototype</span><br/>
<span style="font-size: large;">測試與修正</span><br/>
<span style="font-size: large;"><span style="font-size: small;"><br/></span>
<span style="font-size: small;"><br/></span>
<span style="font-size: small;"><br/></span><span style="font-size: small;">最初階段的重點</span></span><br/>
<span style="font-size: large;">除了一些基本的提問問句以及提問微技術</span><br/>
<span style="font-size: large;">關鍵重點在</span><br/>
<span style="font-size: large;">你的同理心運用的有多好</span><br/>
<span style="font-size: large;">運用的夠好</span><br/>
<span style="font-size: large;">你就能夠取得冰山下的訊息</span><br/>
<span style="font-size: large;">這些訊息可以幫助你得到接近全貌的訊息</span><br/>
<span style="font-size: large;">進而協助你在之後的步驟裡</span><br/>
<span style="font-size: large;">得到更貼近客戶真正需求的設計</span><br/>
<span style="font-size: large;"><span style="font-size: small;"><br/></span>
<span style="font-size: small;"><br/></span>
</span><br/>
<table align="center" cellpadding="0" cellspacing="0" class="tr-caption-container" style="margin-left: auto; margin-right: auto; text-align: center;"><tbody>
<tr><td style="text-align: center;"><a href="https://1.bp.blogspot.com/-bIzS1zdGuI0/XWVTYgmN1XI/AAAAAAAALlA/Onfx5YF_cHg8qkD6uqia0gKL76GbqX8KwCLcBGAs/s1600/IMG_8029.JPG" imageanchor="1" style="margin-left: auto; margin-right: auto;"><span style="font-size: large;"><img border="0" data-original-height="1600" data-original-width="1200" height="640" src="https://1.bp.blogspot.com/-bIzS1zdGuI0/XWVTYgmN1XI/AAAAAAAALlA/Onfx5YF_cHg8qkD6uqia0gKL76GbqX8KwCLcBGAs/s640/IMG_8029.JPG" width="480"/></span></a></td></tr>
<tr><td class="tr-caption" style="text-align: center;"><span style="font-size: large;">認真投入的學員們開心慶祝完成設計實作</span></td></tr>
</tbody></table>
<span style="font-size: large;"><span style="font-size: medium;"><br/></span>
<span style="font-size: medium;">學員們的演練有聲有色</span></span><br/>
<span style="font-size: large;">有好幾位連自己都不相信做出的原型與成果</span><br/>
<span style="font-size: large;"><span style="font-size: medium;"><br/></span>
<span style="font-size: medium;">有角色</span></span><br/>
<span style="font-size: large;">有情境</span><br/>
<span style="font-size: large;">有賣點</span><br/>
<span style="font-size: large;">有道具</span><br/>
<span style="font-size: large;"><span style="font-size: medium;"><br/></span>
<span style="font-size: medium;">有人演出痛點</span></span><br/>
<span style="font-size: large;">有人演出訪談 </span><br/>
<span style="font-size: large;">有人演出idea發想與收歛後的三個</span><br/>
<span style="font-size: large;">甚至有人還演出顧客內心世界聲音XD, 很妙</span><br/>
<span style="font-size: large;">環境, 時間點都清楚呈現</span><br/>
<span style="font-size: large;"><span style="font-size: medium;"><br/></span>
<span style="font-size: medium;">非常精采</span></span><br/>
<span style="font-size: large;"><span style="font-size: medium;"><br/></span>
<span style="font-size: medium;"><br/></span>
<span style="font-size: medium;">幾個其他的思考:</span></span><br/>
<span style="font-size: large;"><br/></span>
<ul>
<li><span style="font-size: large;">你曾經有過怎樣永生難忘的「體驗」嗎? 這個也可以是一種讓自己在發想與體驗相關議題時的一種思考</span></li>
</ul>
<div>
<span style="font-size: large;"><br/></span></div>
<ul>
<li><span style="font-size: large;">關於這方面的主題, 全台灣全世界, 你覺得最可以標竿學習的對象是哪家企業? 會不會他們也曾經有過這個困擾? 他們是如何處理的呢? </span></li>
</ul>
<div>
<span style="font-size: large;"><br/></span></div>
<ul>
<li><span style="font-size: large;">總是會有許多方法, 也都會有一定效果, 我們也都可以回頭想想, 有辦法解決這問題嗎 ?</span></li>
</ul>
<span style="font-size: large;"><br/></span>
<span style="font-size: large;"><span style="font-size: medium;"><br/></span>
<span style="font-size: medium;">整體而言, 這班學員們普遍缺乏一種創意自信, 怯生生地, 不確定自己做的正確與否, 其實也反映了多數的台灣企業同仁們的狀態</span></span><br/>
<span style="font-size: large;"><span style="font-size: medium;"><br/></span>
<span style="font-size: medium;"><a href="https://www.books.com.tw/products/0010641482" target="_blank">《創意自信帶來力量》</a></span></span><br/>
<span style='background-color: white; color: #232323; font-family: "arial" , "helvetica" , sans-serif , "pmingliu";'><span style="font-size: large;">「創意自信」就是相信你有能力改變身邊的世界。創意自信就像肌肉一樣，可藉由鍛鍊與經驗學習變得更強壯。 擁有創意自信的人會做出比較好的選擇、較勇於嘗試新方向，遇到看似棘手的問題也更有能力找到解決方案。</span></span><br/>
<span style="font-size: large;"><span style="font-size: medium;"><br/></span>
<span style="font-size: medium;"><br/></span>
<span style="font-size: medium;">但好消息是</span></span><br/>
<span style="font-size: large;">創意自信是可以鍛鍊的</span><br/>
<span style="font-size: large;"><span style="font-size: medium;"><br/></span>
<span style="font-size: medium;">每個人天生都有</span></span><br/>
<span style="font-size: large;">想像力</span><br/>
<span style="font-size: large;">好奇心 </span><br/>
<span style="font-size: large;"><span style="font-size: medium;"><br/></span>
<span style="font-size: medium;">只是有些人經過社會歷鍊後</span></span><br/>
<span style="font-size: large;">忘的比較徹底一些</span><br/>
<span style="font-size: large;">所以恢復時間需要久一點</span><br/>
<span style="font-size: large;">但仍然有救</span><br/>
<span style="font-size: large;">(如果想救的話^^)</span><br/>
<span style="font-size: large;"><span style="font-size: medium;"><br/></span>
<span style="font-size: medium;"><a href="https://www.hbrtaiwan.com/article_content_AR0002197.html" target="_blank">找回你的創意自信</a></span></span><br/>
<span style="font-size: large;"><span style="font-size: medium;"><br/></span>
<span style="font-size: medium;">祝福大家</span></span><br/>
<span style="font-size: large;">如你所願</span><br/>
<span style="font-size: large;">成為有創意的人</span><br/>
<span style="font-size: large;"><span style="font-size: medium;"><br/></span>
<span style="font-size: medium;">安璐</span></span><br/>
<span style="font-size: large;">108/8/27</span><br/>

</ul>
