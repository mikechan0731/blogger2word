<ul>
<li><b><span style="color: blue; font-size: large;">活動一：關鍵字</span></b></li>
</ul>
<blockquote class="tr_bq">
<span style="font-size: large;">請每一組準備一張右上角寫上組別的便條貼，每位組員把書拿出來，停在封面，請看著封面寫下屬於你自己的關鍵字或圖形，然後拍照上傳到任務群組。結束後，請小組內分享，請前三組分享；說明一下這活動的意義，<b><span style="color: red;">「專屬自己」</span></b>與關鍵字的關聯性。我的關鍵字是ART，圖形是封面的那一波。有的學員寫的是「X」，很特別的答案，其他學員的答案: 模式 / 機會 / ART / GROWTH / OPPORTUNITY / 圖像溝通 / 市場商機 / 創新模式 / 策略創新</span></blockquote>
<ul>
<li><b><span style="color: blue; font-size: large;">活動二：初衷與定錨</span></b></li>
</ul>
<blockquote class="tr_bq">
<span style="font-size: large;">作者群提到他們寫這本書的初衷是「<b><span style="color: red;">老公司該怎麼從組織內部著手，創造新成長策略和業務 ?</span></b> 」 ，剛提到的這句話中「從數千行字裡，畫出<b><span style="color: red;">專屬自己</span></b>，能夠<span style="color: red;"><b>拓展未來</b></span>、<span style="color: red;"><b>翻轉人生</b></span>的一條線」這件事不容易，要如何做到? 就是要能夠「要朝目標、用心思索，才能完成這些事」，重點在掌握目的，商業書不是為了娛樂逍遣，而是為了精進，所以要掌握目的，選哪本書 ? 吸收什麼內容 ? 在哪裡畫線 ? 如何應用 ? <a href="http://lulwechange.blogspot.com/2017/09/blog-post_24.html" target="_blank">一流的人讀書，都在哪裡畫線? ─「喜歡是不需要的」推薦文</a></span></blockquote>
<blockquote class="tr_bq">
<span style="font-size: large;">請大家一人拿一張便條貼，在這張便條貼上寫下你個人看這本書的初衷/目的，分組拍照上傳，小組分享，分享後，請大家<b><span style="color: blue;">把自己的初衷目的化成一句提問</span></b>，一句想從書中找到答案的提問 ，例如: 「希望有學到創新方法」化成一提問「創新方法如何做?」「商業如何創造價值?」……其他學員的初衷是: 思考方法 / 學習新的東西 / 商模創新 / 改變公司成長 / 學習 /改變/ 改變公司策略 / 系統性方法引出機會 / 求新求進步 / 想要了解如何開拓客戶 / 看未來創新趨勢 / 找出新的商業模式 / 未來市場 / 授課須求 /靈感 / 突破瓶頸 / 對工作有幫助 /改變想法 / 創造新的商業模式 / 學習圖像式溝通 / 學習創新模式 / 學商業創新模式 / 找新IDEA  / 創新想法 / 找創新模式找機會</span></blockquote>

</blockquote>

<br/>
<blockquote class="tr_bq">
<span style="font-size: large;">請大家翻到P17的讀書旅程，這本書運用了三個重要元素，其一就是圖像溝通，所以它的架構也用了讀書旅程的圖像溝通來進行，首先請大家看到它的三個顏色，請對照到P204的三個階段與顏色，紅色代表初步階段，藍色代表演進階段，綠色代表擴散階段，再請大家看一下書的側面，都有著三種顏色，而三種顏色也有不同的呈現方式，有的是一整區塊的，有的有兩個短區塊，有的有一個短區塊，這<b>本書很討厭的地方是它很重，但它之所以這麼重，就是因為它很看重「圖像溝通」，所以它需要用全彩的方式呈現它的內容</b>，所以邀請大家來閱讀這樣的巧思呈現。 </span></blockquote>
<blockquote class="tr_bq">
<span style="font-size: large;">再來，我們回到 P17的讀書旅程，它有<b><span style="color: red;">5個小圖像</span></b>，分別代表了它所說的五個元素組成 ( <b><span style="color: red;">核心觀念 / 啟發案例 /靈感火花/ 活動/ 範本</span></b>) ，請大家在書中的「建構營收模式」這裡，運用分組討論，把這5個小圖像對照的五個元素名稱寫在旁邊，討論結束後拍照上傳到任務群組。這活動結束後，跟大家分享「看到模式」這件事</span></blockquote>
<blockquote class="tr_bq">
<span style="font-size: large;">再來，同樣是P17的讀書旅程，中間的三個階段，發現你的新成長機會／打造策略／推動業務，請大家把<b><span style="color: red;">「發現／打造／推動」</span></b>圈起來，舉手搶答，這是什麼詞性?  然後說明它跟下面小項方法的關聯性，再來是「範本」，再舉手搶答，這是什麼詞性?  然後說明它跟這本書主題的關聯性</span></blockquote>
<br/>
<ul>
<li><span style="font-size: large;"><b><span style="color: blue;">結尾：</span></b>請大家翻下一頁，「創新領我們前行！」同時說明這本書特色之一，經常有這種兩頁串在一起的大字頁，這也是圖像溝通的一部份，邀請大家一起唸，共同祝福與結尾</span></li>
</ul>
