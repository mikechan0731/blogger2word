<h3 class="post-title entry-title" itemprop="name">
臨時手術住院需備物品
</h3>
<div class="post-header">
<div class="post-header-line-1"></div>
</div>
<div class="post-body entry-content" id="post-body-440222515088101794" itemprop="description articleBody">
<table align="center" cellpadding="0" cellspacing="0" class="tr-caption-container" style="margin-left: auto; margin-right: auto; text-align: center;"><tbody>
<tr><td style="text-align: center;"><a href="https://1.bp.blogspot.com/-KNnQZmM-0Q0/VvlCHtUTFRI/AAAAAAAAJYU/4r9ur4LJRI8Zpf5fIE3Tq_B8mYKit0shw/s1600/12705336_10153480767193715_4983995271993329774_n.jpg" imageanchor="1" style="margin-left: auto; margin-right: auto;"><span style="font-size: large;"><img border="0" height="640" src="https://1.bp.blogspot.com/-KNnQZmM-0Q0/VvlCHtUTFRI/AAAAAAAAJYU/4r9ur4LJRI8Zpf5fIE3Tq_B8mYKit0shw/s640/12705336_10153480767193715_4983995271993329774_n.jpg" width="480"/></span></a></td></tr>
<tr><td class="tr-caption" style="text-align: center;"><span style="font-size: large;">照進病房的陽光，很美好~</span></td></tr>
</tbody></table>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">突如其來的開刀手術安排</span><br/>
<span style="font-size: large;">先生門診後的</span><span style="font-size: large;">隔兩日</span><br/>
<span style="font-size: large;">就要連住三天</span><a name="more"></a><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">原本已因大女兒B型流感</span><br/>
<span style="font-size: large;">而大為打亂的行程</span><br/>
<span style="font-size: large;">又繼續亂成一團</span><br/>
<span style="font-size: large;">延的延</span><br/>
<span style="font-size: large;">取消的取消</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">等到結束行程趕到醫院時</span><br/>
<span style="font-size: large;">發現自己準備得</span><br/>
<span style="font-size: large;">不是很充足</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">其實不需帶太多東西</span><br/>
<span style="font-size: large;">但若有下次</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">希望可加速準備時程</span><br/>
<span style="font-size: large;">所以簡記所需物品如下</span><br/>
<span style="font-size: large;">(陪住人)</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">拖鞋</span><br/>
<span style="font-size: large;">牙刷、牙膏、洗髮精、潤髮精</span><br/>
<span style="font-size: large;">換洗衣物</span><br/>
<span style="font-size: large;">充電器</span><br/>
<span style="font-size: large;">隨身藥品</span><br/>
<span style="font-size: large;">書、筆記</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">(住院人)</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">拖鞋</span><br/>
<span style="font-size: large;">牙刷、牙膏、肥皂</span><br/>
<span style="font-size: large;">換洗衣物</span><br/>
<span style="font-size: large;">充電器</span><br/>
<span style="font-size: large;">臉盆</span><br/>
<span style="font-size: large;">健保卡、提款卡</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">整體而言</span><br/>
<span style="font-size: large;">覺得很幸運</span><br/>
<span style="font-size: large;">很幸運地雖然遇到這麼多的事</span><br/>
<span style="font-size: large;">但都是有種逢兇化吉的</span><br/>
<span style="font-size: large;">輕微事件</span><br/>
<span style="font-size: large;">感恩</span><br/>
<span style="font-size: large;">感恩</span><br/>
<span style="font-size: large;">感恩</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">祝福</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">心安圓滿</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">安璐</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">105/3/28</span>
