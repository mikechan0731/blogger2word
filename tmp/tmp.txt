<h3 class="post-title entry-title" itemprop="name">
學歷證照不是最重要的，自我經驗與成長經歷才是
</h3>
<div class="post-header">
<div class="post-header-line-1"></div>
</div>
<div class="post-body entry-content" id="post-body-2068836482854950474" itemprop="description articleBody">
<div class="text_exposed_root text_exposed">
<span style="font-size: large;">「站在語言教學專業的立場，</span></div>
<div class="text_exposed_root text_exposed">
<span style="font-size: large;">長期以來我發現臺灣這一類文章，</span></div>
<div class="text_exposed_root text_exposed">
<span style="font-size: large;">潛在著一個迷思：過度的自卑與自信。</span><a name="more"></a><br/></div>
<div class="text_exposed_root text_exposed">
<span style="font-size: large;"></span> </div>
<div class="text_exposed_root text_exposed">
<span style="font-size: large;"> 我想說的是，其實站在語言專業的角度來看，</span></div>
<div class="text_exposed_root text_exposed">
<span style="font-size: large;">在非母語的環境工作，所需要的語言條件，</span></div>
<div class="text_exposed_root text_exposed">
<span style="font-size: large;">其實並不是一個「感覺的問題」，</span></div>
<div class="text_exposed_root text_exposed">
<span style="font-size: large;">不是自己覺得還不夠或者自己覺得很不錯就可以自己憑空認定的。</span></div>
<div class="text_exposed_root text_exposed">
<span style="font-size: large;"> 語言能力的認定，其實是一個「事實的撰寫」，</span></div>
<div class="text_exposed_root text_exposed">
<span style="font-size: large;">一個「自我學習經歷的事實陳述」。</span></div>
<div class="text_exposed_root text_exposed">
<span class="text_exposed_hide"><span style="font-size: large;">⋯⋯</span></span></div>
<div class="text_exposed_root text_exposed">
<span class="text_exposed_hide"><span style="font-size: large;"></span></span> </div>
<div class="text_exposed_show">
<span style="font-size: large;"> 也就是說，一開始要在國外申請履歷的時候</span><br/>
<span style="font-size: large;">（特別是歐洲這種多語言地區），</span><br/>
<span style="font-size: large;">要把自己所有學習外語的種類，過程，</span><br/>
<span style="font-size: large;">好的與壞的，通通說，老實說。</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;"> 比如，一個台灣二十五歲的年輕人，</span><br/>
<span style="font-size: large;">可能的語言經歷是這樣的：</span><br/>
<span style="font-size: large;"> 母語流利程度：台語（口語流利但不善書寫），</span><br/>
<span style="font-size: large;">普通話（聽說讀寫流利）。</span><br/>
<span style="font-size: large;"> 第一外語：英語 （從國中學起，閱讀能力中高級，口語方面，生活基本溝通沒問題）</span><br/>
<span style="font-size: large;"> 第二外語：日語 （口語初級，閱讀方面能看懂菜單和簡單日語廣告或產品說明）</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;"> 以上，注意到沒？我寫的是一個人學習語言的自傳，</span><br/>
<span style="font-size: large;">裡面並不一定會有檢定甚至任何成績的證明，</span><br/>
<span style="font-size: large;">也不一定樣樣能力完美頂尖，</span><br/>
<span style="font-size: large;">但，這就是一個讓你未來雇主，</span><br/>
<span style="font-size: large;">一個認識你多語能力的機會。</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;"> 語言的學習，從來不是徒勞無功的，</span><br/>
<span style="font-size: large;">它一定走過必留痕跡，而且是美好的足跡。</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">那為什麼不應該這樣寫呢？</span><br/>
<span style="font-size: large;"> 我一直覺得，在語言學習上，</span><br/>
<span style="font-size: large;">台灣人很容易預設一個偏離事實的立場，</span><br/>
<span style="font-size: large;">不是覺得自己太差，就是覺得自己太強。</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">但明明語言，</span><br/>
<span style="font-size: large;">不是一個自我感覺的問題，</span><br/>
<span style="font-size: large;">而是人與人之間，</span><br/>
<span style="font-size: large;">生活溝通始終存在的通路，</span><br/>
<span style="font-size: large;">一個心與心的地圖。</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">你只有真誠地面對它的來處與座標，</span><br/>
<span style="font-size: large;">才能知道下一條路，該怎麼走。</span><br/>
<span style="font-size: large;"> 肯定自己所有的，繼續自己未有的。</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;"> 補記：</span><br/>
<span style="font-size: large;"> 第一次看到歐洲人寫的履歷時，非常震撼。</span><br/>
<span style="font-size: large;">覺得這一些老歐怎麼這麼老實（承認自己德語閱讀流利但口語爆爛），</span><br/>
<span style="font-size: large;">又這麼敢寫（明明沒上過義大利文課，就不要臉地說自己會用義大利語買東西）。</span><br/>
<span style="font-size: large;"> 後來在語言教學研究所，</span><br/>
<span style="font-size: large;">發現一個人的語言過程其實是一個非常有趣、多元而複雜的心靈自傳，</span><br/>
<span style="font-size: large;">一個屬於自己的故事 (biographie langagière)。</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">而那個「自己的故事」，雖然不見得完美，</span><br/>
<span style="font-size: large;">但也因為平凡、真實而顯得更有力量。</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;"> 重視形式的升學主義、淘汰篩選的菁英制度，</span><br/>
<span style="font-size: large;">容易讓人流於相信表面的東西，</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">比如學歷，比如文憑，</span><br/>
<span style="font-size: large;">比如成功者的分享與模仿，</span><br/>
<span style="font-size: large;">而往往忽略了真實存在、</span><br/>
<span style="font-size: large;">發生過的自我經驗與成長經歷。」</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">這是在</span><a href="https://www.facebook.com/syhjjr?fref=ts" target="_blank"><span style="font-size: large;">黃世宜老師FB頁面</span></a><span style="font-size: large;">上看到的文章</span><br/>
<span style="font-size: large;">經她同意轉載</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">她文中所提的文章</span><br/>
<span style="font-size: large;">是指天下雜誌的一篇文章〈</span><a href="http://blog.cw.com.tw/blog/profile/237/article/1912#comment-30460" target="_blank"><span style="font-size: large;">在美國工作：英文到底要有多好</span></a><span style="font-size: large;">〉</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">很喜歡她文中所提概念</span><br/>
<span style="font-size: large;">相當精闢</span><br/>
<span style="font-size: large;">也是自己平常在擔任職涯發展教練</span><br/>
<span style="font-size: large;">進行職涯晤談時</span><br/>
<span style="font-size: large;">協助沒有自信的來談者</span><br/>
<span style="font-size: large;">在重新建構自我生涯地圖時</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">著力的方向與方式</span><br/>
<span style="font-size: large;">很有共鳴</span><br/>
<span style="font-size: large;">特別轉載於此</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">「語言，</span><br/>
<span style="font-size: large;">不是一個自我感覺的問題，</span><br/>
<span style="font-size: large;">而是人與人之間，</span><br/>
<span style="font-size: large;">生活溝通始終存在的通路，</span><br/>
<span style="font-size: large;">一個心與心的地圖。</span><br/>
<span style="font-size: large;"> </span><br/>
<span style="font-size: large;">你<strong>只有真誠地面對它的來處與座標，</strong></span><br/>
<strong><span style="font-size: large;">才能知道下一條路，該怎麼走。</span></strong><br/>
<span style="font-size: large;"><strong> 肯定自己所有的，繼續自己未有的</strong>。」 ~黃世宜</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">「一個人的語言過程其實是一個非常有趣、多元而複雜的心靈自傳，</span><br/>
<span style="font-size: large;">一個屬於自己的故事 (biographie langagière)。</span><br/>
<span style="font-size: large;"> </span><br/>
<span style="font-size: large;">而那個『自己的故事』，雖然<strong>不見得完美，</strong></span><br/>
<span style="font-size: large;"><strong>但也因為平凡、真實而顯得更有力量</strong>。」 ~黃世宜</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">也很像是我在為職涯晤談來談者</span><br/>
<span style="font-size: large;">做能力盤點與未來職能發展計劃的</span><br/>
<span style="font-size: large;">能力強項卡排卡時</span><br/>
<span style="font-size: large;">的中心信念</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">很謝謝世宜老師</span><br/>
<span style="font-size: large;">幫我清楚地說明出來</span><br/>
<span style="font-size: large;">很是喜悅與振奮</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">學歷證照不會是最重要的</span><br/>
<span style="font-size: large;">讓我們一起剝開表相</span><br/>
<span style="font-size: large;">重新看到</span><br/>
<span style="font-size: large;">本來存有在自己身上的</span><br/>
<span style="font-size: large;">自我經驗與成長經歷的</span><br/>
<span style="font-size: large;">美好價值</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">祝福大家</span><br/>
<span style="font-size: large;">享有重新看到</span><br/>
<span style="font-size: large;">自己的美好</span><br/>
<span style="font-size: large;"></span><br/>
<span style="font-size: large;">安璐</span><br/>
<span style="font-size: large;">105/1/</span><br/>
